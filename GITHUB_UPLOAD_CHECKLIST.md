# GitHub upload checklist

Upload these items to the top level of the repository:

- `.env.example`
- `.gitignore`
- `.npmrc`
- `README.md`
- `SUPERVISOR_NOTES.md`
- `GITHUB_UPLOAD_CHECKLIST.md`
- `package.json`
- `package-lock.json`
- `server.js`
- `vercel.json`
- the complete `public` folder
- the complete `scripts` folder

Do not upload:

- `.env`
- `node_modules`
- your OpenAI API key
- your Vercel account credentials

After uploading, the GitHub repository home page should show `server.js`, `package.json`, and `vercel.json` immediately—not inside another enclosing folder.
